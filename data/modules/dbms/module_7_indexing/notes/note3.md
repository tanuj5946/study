### B+ Tree Indexing

B+ Tree is a balanced tree structure used in databases.

Characteristics:

- All leaf nodes at same level.
- Internal nodes store keys.
- Leaf nodes store data pointers.
- Leaf nodes are linked.

Advantages:

- Efficient search
- Efficient insertion
- Efficient deletion
- Supports range queries

Difference between B-Tree and B+ Tree:

B-Tree:
- Data stored in internal nodes.

B+ Tree:
- Data stored only in leaf nodes.
