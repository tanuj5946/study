### Failure and Recovery Concepts

Failure can occur due to:
- Transaction failure
- System crash
- Disk failure
- Power failure

Recovery ensures database consistency after failure.

Types of Failures:

1. Transaction Failure
   Caused by logical error or deadlock.

2. System Crash
   Power failure or hardware crash.

3. Media Failure
   Disk crash or data corruption.

Recovery Techniques:
- Log-based recovery
- Checkpointing
- Shadow paging

Exam Notes:
- Types of failure are frequently asked.
- Define recovery clearly.
