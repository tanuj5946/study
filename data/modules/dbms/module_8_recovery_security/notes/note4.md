### Database Security

Database security protects data from unauthorized access.

Security Goals:

1. Confidentiality
2. Integrity
3. Availability

Authentication:
Verifies identity of user.

Authorization:
Determines access rights.

Access Control Methods:

1. Discretionary Access Control (DAC)
2. Mandatory Access Control (MAC)
3. Role-Based Access Control (RBAC)

SQL Security Commands:

GRANT:
Provides privileges.

REVOKE:
Removes privileges.

Example:
GRANT SELECT ON Student TO user1;

Encryption:
Protects data from unauthorized viewing.

Exam Notes:
- Difference between authentication and authorization.
- GRANT and REVOKE syntax important.
