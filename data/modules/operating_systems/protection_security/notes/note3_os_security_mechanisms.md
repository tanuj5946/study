### OS Security Mechanisms

Operating systems implement mechanisms to protect resources and data.

Protection Domains:
- Define boundaries of access rights.
- Separate user and kernel modes.

Common security threats:
- Malware
- Viruses
- Worms
- Trojan horses

Security mechanisms include:
- Encryption
- Firewalls
- Intrusion detection systems

OS security ensures confidentiality, integrity, and availability.
