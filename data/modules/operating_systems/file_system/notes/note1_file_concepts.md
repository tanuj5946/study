### File Concepts

A file is a collection of related information stored on secondary storage.

A file system provides a mechanism for storing and accessing data on storage devices.

File attributes include:
- Name
- Identifier
- Type
- Location
- Size
- Protection
- Time and date information

Operations performed on files:
- Create
- Open
- Read
- Write
- Close
- Delete

File systems ensure data organization, protection, and efficient access.

---

Exam Notes:

- A file is a logical storage unit.
- File attributes are commonly asked in exams.
- File operations define how users interact with files.
